This is simple calculator for probability of an event after certain retries. E.g. when we flip coin two times and want to see heads, on second try probability of success will be 0.75 (theoretically with ideal coin)

How to use:

  1. Enter the target probability value (if P(A)â‰¥Pt calculation will stop)
  2. Enter one or multiple initial P values. For each iteration program uses next available value and if there is no such values left, it will use last available one. (E.g. if you provide values 0.1, 0.4 0.1 will be used for first try and 0.4 for second and all remaining tries)
  3. Press button and wait

Note: this program was not developed for very hard calculations. It uses one background worker thread and generates table row for each try.
